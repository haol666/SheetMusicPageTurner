# 智能面部识别乐谱自动翻页工具

## 项目简介

本项目是一款基于iOS平台的智能面部识别乐谱自动翻页工具，旨在帮助钢琴演奏者实现“解放双手”的演奏体验。通过平板电脑的前置摄像头捕捉演奏者的面部特征，利用面部识别技术（如识别张嘴、点头等特定动作）来触发翻页指令，实现无接触式的乐谱切换，提升演奏的专业性与便捷性。

## 功能特性

### 核心功能

1. **面部识别翻页控制**
   - 支持通过张嘴动作触发翻页
   - 支持通过点头动作触发翻页
   - 动作灵敏度可调节

2. **乐谱查看器**
   - 支持PDF和多图片格式的乐谱展示
   - 支持页面平滑切换、缩放及亮度调节
   - 手动翻页功能

3. **文件管理**
   - 支持从本地文件夹或云端导入乐谱文件
   - 支持对已导入乐谱进行分类、删除、搜索

## 技术实现

- **开发语言**：Swift 5.0
- **最低支持系统**：iOS 12.0
- **核心框架**：
  - Vision Framework - 用于面部识别
  - AVFoundation - 用于摄像头捕获
  - UIKit - 用于用户界面

## 安装与使用

### 安装

1. 克隆项目到本地
2. 使用Xcode打开 `SheetMusicPageTurner.xcodeproj` 文件
3. 选择目标设备（建议使用iPad）
4. 运行项目

### 使用方法

1. **首次启动**：应用会请求摄像头权限，请允许访问前置摄像头
2. **导入乐谱**：通过文件管理功能导入PDF或图片格式的乐谱
3. **选择翻页动作**：在设置中选择需要的翻页触发动作（张嘴或点头）
4. **开始使用**：进入乐谱查看模式，通过预设动作触发翻页

## 核心代码说明

### 面部识别实现

项目使用iOS 12的Vision Framework实现面部识别，主要包括以下功能：

- **张嘴检测**：通过计算嘴部的高宽比（MAR）来判断是否张嘴
- **点头检测**：通过分析头部的垂直运动来判断是否点头
- **动作触发**：当检测到预设动作时，执行翻页操作

### 翻页控制

- **自动翻页**：通过面部识别触发
- **手动翻页**：通过点击屏幕按钮触发
- **防重复触发**：设置1秒冷却时间，避免连续触发

## 注意事项

1. 使用时请确保光线充足，以获得最佳的面部识别效果
2. 建议将设备固定在合适的位置，确保面部在摄像头的视野范围内
3. 首次使用时可能需要调整动作灵敏度，以适应个人习惯
